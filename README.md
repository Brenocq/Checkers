## Installation
First, you need to download the OpenGl and Glut:
```
$ sudo apt-get install mesa-common-dev
$ sudo apt-get install freeglut3-dev

$ git clone https://github.com/Brenocq/Checkers.git
```

## Compilling and running
```
$ cd Checkers 
$ make clean
$ make

$ ./run
```
